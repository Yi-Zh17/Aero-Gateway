# Agent context

This dashboard was implemented by Claude Code as part of the Aero Gateway project. The UI in `app/page.tsx` was generated by the agent based on the Go admin API contract defined during development.

## What the agent implemented

- `app/page.tsx` — full dashboard page: plugin upload card, live metrics card, error states
- `app/layout.tsx` — stripped to essentials, removed default Next.js boilerplate

## What the agent did NOT implement

- The Go proxy, admin server, and hot-swap logic (`proxy/`)
- The Rust WebAssembly filter plugin (`plugin/`)
- System architecture and concurrency design

## Next.js version note

This project uses Next.js 16 (App Router). APIs and conventions differ from older versions. Read `node_modules/next/dist/docs/` before making changes.
